### Краткое описание

Минималистичный сетап редактора кода vscode. Данный сетап нацелен на фокусировки на код, нечего лишнего. Все доп вкладки управляются с помощью клавиатуры что делает работу более быстрой. Также данный сетап довольно красивый. Отключены лишние функции и визуальные элементы которые только мешают. Но добавлены полезные функции например мульти-курсор. Все настройки разделены по разделам поэтому будет легко их кастомизировать. 

### Сочитания клавиш

Как было написано выше почти все окна управляются через клавиатуру, да по началу это будет казатся неудобным но если привыкнуть все станет намного быстрее, в настройке нету файла с сочитанием клавиш ведь почти все дефолтное. Ниже список основных сочитаний 

- **⌥ + Space** - Палитра команд *дефолт: ⌘ + Shift  + P*
- **⌘ + P** - Удобный поиск файлом в текущей директории
- **⌘ + F** - Искать по слову в коде
- **⌘ + Shift + O** - Переход к функции/переменной текущего файла
- **⌘ + Shift + E** - Вкладка с файлами
- **⌘ + J** - Открыть/скрыть панель терминала  *дефолт: ⌘ + ,*
- **⌘ + b** - Открыть/скрыть боковую панель
- **⌃ + Tab** - alt-tab с файлами 
- **⌘ + W** - Закрыть текущий файл
- **⌃ + Shift + G** - Работа с Git
- **⌘ +⌥ + b** - Открыть/скрыть панель с ии чатом 
- **⌘ + Shift + X** - Вкладка с расширениями 
- **⌘ + Shift + E** - Открыть панель проблем в коде

Этих сочитаний хватит для продуктивной работы.

### Про языки программирования и кастомизация

Я программирую на python, и поэтому некоторые настройки, например линтер или настройки расширений заточены именно под этот язык. Это решается удалением разделов которые вам не нужны. Сочитания клавиш описаны для мак но если у вас windows или linux расстраиватся не стоит ведь у вас уже все настроено по умолчанию, Клавиша Command **⌘** - Control и тд.
